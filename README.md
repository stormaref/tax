# محاسبه‌گر مالیات بر حقوق | Iran Salary Tax Calculator

Online progressive salary tax calculator for Iranian tax years **1404** and **1405**. Enter your taxable salary in Toman and instantly see your tax, social security insurance (7%), and net take-home pay.

Inspired by [majidh1/iranian-salary-tax](https://github.com/majidh1/iranian-salary-tax).

**Live demo:** [stormaref.github.io/tax](https://stormaref.github.io/tax/)

## Features

- Progressive multi-bracket tax calculation for 1404 & 1405
- Real-time results as you type
- 7% social security insurance deduction
- Expandable tax bracket reference table
- Persian digit and currency formatting
- Responsive, mobile-friendly UI
- Zero dependencies — single HTML file, no build step

## Usage

```bash
git clone https://github.com/stormaref/tax.git
cd tax
open index.html
```

Or visit the [live demo](https://stormaref.github.io/tax/) directly.
